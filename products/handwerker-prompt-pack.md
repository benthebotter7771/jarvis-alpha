# Handwerker Prompt-Pack - Professionelle KI-Assistenz für Handwerksbetriebe

## Überblick
Dieses Prompt-Pack enthält spezialisierte KI-Prompts für Handwerker (Elektriker, Installateure, Maler, Zimmerer) zur Optimierung von täglichen Arbeitsabläufen, Kundenkommunikation und Dokumentation.

## Zielgruppe
- Kleine und mittlere Handwerksbetriebe
- Selbstständige Handwerker
- Handwerksmeister mit Mitarbeitern

## Preis: €49 (Einmalig) oder €9/Monat (Abonnement)

## Prompt-Kategorien

### 1. Kundenkommunikation
**Angebotserstellung**
```
Erstelle ein professionelles Angebot für folgende Leistungen:
- Kunde: [Name, Adresse]
- Leistungen: [Liste der Arbeiten]
- Materialkosten: €[Betrag]
- Arbeitszeit: [Stunden]
- Frist: [Datum]
- Besondere Bedingungen: [Text]

Format: Professionelle PDF-Vorlage mit Logo-Platzhalter, klare Aufschlüsselung, rechtliche Hinweise.
```

**Kunden-E-Mail-Vorlagen**
```
Verfasse eine freundliche, professionelle E-Mail an Kunden [Name] zum Thema [Thema: Terminbestätigung, Rechnungserinnerung, Projektupdate, Problemlösung]. Tonfall: zuvorkommend aber direkt.
```

### 2. Dokumentation & Berichte
**Arbeitsbericht Generator**
```
Erstelle einen detaillierten Arbeitsbericht für [Projektname] mit:
- Durchgeführte Arbeiten
- Verwendete Materialien
- Arbeitszeiten pro Mitarbeiter
- Besondere Vorkommnisse
- Empfehlungen für Folgearbeiten

Format: Tabellarisch, maschinenlesbar für Buchhaltung.
```

**Baustellen-Tagebuch**
```
Führe ein Baustellen-Tagebuch für [Datum]:
- Wetterbedingungen
- Anwesende Mitarbeiter
- Fortschritt (%)
- Probleme/Verzögerungen
- Sicherheitschecks
```

### 3. Kalkulation & Planung
**Materialkalkulation**
```
Berechne Materialbedarf für [Projekttyp: z.B. Badezimmersanierung]:
- Fläche/Raummaße: [Angaben]
- Materialien: [Liste]
- Verschnitt: [%]
- Gesamtkosten inkl. MwSt.
- Lieferantenvergleich (3 Optionen)
```

**Zeitplanung**
```
Erstelle einen realistischen Zeitplan für [Projekt] mit:
- Meilensteinen
- Kritischem Pfad
- Pufferzeiten
- Abhängigkeiten zwischen Gewerken
```

### 4. Rechtliches & Verwaltung
**AVV-Formular (Auftragsbestätigung)**
```
Generiere ein AVV-Formular gemäß VOB mit:
- Leistungsbeschreibung
- Preisen
- Zahlungsbedingungen
- Gewährleistung
- Haftungsausschlüssen
```

**Rechnungserstellung**
```
Erstelle eine rechtskonforme Rechnung nach GoBD mit:
- Fortlaufende Rechnungsnummer
- USt-ID
- Leistungsbeschreibung §14 UStG
- Fälligkeitsdatum
- Zahlungsinformationen
```

### 5. Marketing & Akquise
**Kundenbewertungsanfrage**
```
Verfasse eine höfliche Nachricht zur Bewertungsabfrage für [Plattform: Google, Facebook, MyHammer] nach erfolgreichem Abschluss von [Projekt].
```

**Social Media Content**
```
Erstelle 3 Social-Media-Posts für [Plattform] zum Thema [Fachgebiet] mit:
- Fachwissen-Tipp
- Vorher/Nachher-Bildbeschreibung
- Kundenstimme
```

## Bonus-Prompts
**Notfall-Checkliste**
```
Erstelle eine Checkliste für [Notfalltyp: Wasserschaden, Stromausfall, Sicherheitsrisiko] mit:
- Sofortmaßnahmen
- Kontakte (Versicherung, Notdienst)
- Dokumentationspflichten
```

**Mitarbeiterschulung**
```
Entwickle ein 10-minütiges Schulungsskript zum Thema [Sicherheit, neue Technik, Kundenservice] für Handwerksgesellen.
```

1. **Elektriker**: Angebote für Smart-Home-Installationen
2. **Installateur**: Heizungsoptimierung mit Förderungsberatung
3. **Maler**: Farbberatung mit 3D-Visualisierung
4. **Zimmerer**: Dachsanierung mit Statik-Check
